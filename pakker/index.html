Pakker

Pakker eller bibliotek er en sammling python script som gir deg tilgang til en gitt type funksjonalitet. Det kan sees på som et bestemt verktøy som lar deg løse et problem.

Eksempel

	# importerer pakken random
	import random
	r = random.randint(1,6)
	print r

Utskrift

	"Ett tilfelldig tall mellom 1 og 6"
