Objekter - Metoder

Funksjoner som er tilordnet et objekt kalles en metode. Første argument til en metode i python er en referanse til seg selv (self). Denne variabelen brukes for å nå atributter knyttet til objektet. Dette skiller seg fra programmeringsspråk som C++ og Java!

Eksempel

# Definerer en klasse sirkel
class Sirkel:
   radius = 1

   # metode skriver ut radiusen
   def printRadius(self):
      print self.radius

   # setter radius
   def settRadius(self, r):
      self.radius = r

s1 = Sirkel()
s1.settRadius(42)
s1.printRadius()
