Lister

En liste i Python er noe som kan inneholde flere elementer. Elementene behøver ikke å være av samme type så du kan lagre forskjellige type verdier sammen. For å lage en liste setter du listenavnet lik flere elementer etter hverandre separert med komma i mellom og klammer på hver side.

Eksempel


	list1 = ['katt', 5, 'fredag', 7, 2014];
        list2 = [1,2,3,4,5];

	print "List 1:", list1 
        print "List 2:", list2

Utskrift

	List 1: ['katt', 5, 'fredag', 7, 2014]
        List 2: [1,2,3,4,5]
